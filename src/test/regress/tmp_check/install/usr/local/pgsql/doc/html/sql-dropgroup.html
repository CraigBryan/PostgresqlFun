DROP GROUP

Name

DROP GROUP -- remove a database role

Synopsis

DROP GROUP name [, ...]

Description

DROP GROUP is now an alias for DROP ROLE, which see for more information.

Compatibility

There is no DROP GROUP statement in the SQL standard.
