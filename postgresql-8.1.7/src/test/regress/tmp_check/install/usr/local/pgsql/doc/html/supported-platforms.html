14.7. Supported Platforms

PostgreSQL has been verified by the developer community to work on the platforms listed below. A supported platform generally means that PostgreSQL builds and installs according to these instructions and that the regression tests pass. "Build farm" entries refer to active test machines in the PostgreSQL Build Farm. Platform entries that show an older version of PostgreSQL are those that did not receive explicit testing at the time of release of version 8.1 but that we still expect to work.

Note: If you are having problems with the installation on a supported platform, please write to <pgsql-bugs@postgresql.org> or <pgsql-ports@postgresql.org>, not to the people listed here.
